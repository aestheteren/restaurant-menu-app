# Restaurant Menu & Order Application

## Overview
Restaurant Menu & Order Application is an Android mobile application developed to provide a simple digital ordering experience for restaurants.

Users can browse menu items, view product details, add items to a cart, and place orders easily through a user-friendly interface.

---

## Features
- Browse restaurant menu items
- View detailed product information
- Add and remove items from the cart
- Automatically updated total price
- Place orders digitally

---

## Technologies Used
- Java
- Android Studio
- XML
- RecyclerView
- Firebase Cloud Firestore

---

## Project Structure
- Menu screen for listing products
- Product detail screen
- Cart screen with dynamic total price calculation
- Firebase integration for order storage
